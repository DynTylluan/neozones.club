Font Front

This is the folder that stores all of the fonts used on the general part of the NeoZones site. In case you are wondering, the main font used at NeoZones is Drogowskaz, as made by the Polish government. The current copyright for the font is that it is "Freeware; for non-commercial use". You can download the font from this URL: http://eu07.pl/forum/index.php?action=dlattach;topic=10206.0;attach=1611.

If you are logged into NeoZones, you can access the folder page via this URL: https://neocities.org/dashboard?dir=%2Fcommon%2Ffonts
